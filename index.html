Sistema de control de velocidad (PWM)

Diseño del puente H con MOSFETs adecuados para 3V–50V, control por ESP32, implementación de PID básico para control de velocidad, monitoreo de temperatura y RPM, y documentación completa con esquemas, firmware y archivos Gerber.

Robot evasor de obstáculos

Robot autónomo con sensor ultrasónico HC-SR04, control por microcontrolador, lógica de evasión, pruebas de rendimiento, vídeo demostrativo y código comentado listo para entrega académica.

Sistema de riego con energía renovable

Integración de panel solar, controlador de carga, válvulas eléctricas, telemetría básica y simulación del consumo energético. Documentación técnica y guía de montaje.
